# Klotski HTML 版本

Thanks for https://github.com/jojonium/Klotski

这是原Java Klotski拼图游戏的HTML/JavaScript版本，保持了所有原有功能并提供了现代化的用户界面。

## 功能特性

### 游戏功能
- ✅ 完整的Klotski拼图游戏逻辑
- ✅ 4种不同的起始配置
- ✅ 方块选择和移动
- ✅ 胜利条件检测
- ✅ 步数统计

### 用户界面
- 🎨 现代化的渐变设计
- 📱 响应式布局，支持移动设备
- 🖱️ 鼠标拖拽移动方块
- ⌨️ 键盘快捷键支持
- 🎯 直观的方向控制按钮

### 游戏控制
- **鼠标操作**：点击选择方块，拖拽移动
- **键盘控制**：
  - 方向键：移动选中的方块
  - Ctrl+R：重置游戏
  - Ctrl+S：保存游戏
  - Ctrl+O：加载游戏
  - Ctrl+1/2/3/4：切换配置

### 保存/加载
- 使用浏览器本地存储保存游戏状态
- 可以随时保存和恢复游戏进度

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```

## 游戏规则

1. **目标**：将红色的大型2x2方块从棋盘底部的开口处移出
2. **操作**：
   - 点击方块选择
   - 使用方向键或方向按钮移动
   - 也可以拖拽方块移动
3. **限制**：只有大型2x2方块可以离开棋盘
4. **胜利**：成功将主方块移出棋盘

## 技术实现

### 架构
- **模型层**：`Piece` 和 `Board` 类处理游戏逻辑
- **视图层**：HTML/CSS 提供用户界面
- **控制器层**：JavaScript 事件处理用户交互

### 核心类
- `Piece`：表示单个方块，包含位置、尺寸和移动逻辑
- `Board`：管理整个游戏状态，包括方块数组、选择状态、移动验证等

### 特性
- 完全响应式设计
- 现代化的CSS渐变和动画
- 本地存储支持
- 完整的键盘和鼠标控制
- 拖拽移动支持
