# 华容道游戏

一个经典的华容道（Klotski）滑动方块益智游戏。

## 游戏特色

- 🎮 4种不同的游戏布局
- 💾 保存和加载游戏进度
- 📱 响应式设计，支持移动设备
- ⌨️ 键盘和鼠标控制
- 🎯 步数统计和游戏状态显示

## 如何游戏

1. 目标：将红色2x2方块移动到下方的出口
2. 控制方式：
   - 鼠标拖拽方块
   - 方向键移动
   - 屏幕上的方向按钮
3. 快捷键：
   - `Ctrl+R`: 重新开始
   - `Ctrl+S`: 保存游戏
   - `Ctrl+O`: 加载游戏
   - `Ctrl+1` 到 `Ctrl+4`: 切换布局

## 开始游戏

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术栈
